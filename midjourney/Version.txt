<h1>Version</h1>
</div>
<hr /></div>
<div class="content_block_text" dir="auto">
<h4>Midjourney routinely releases new model versions to improve efficiency, coherency, and quality. The latest model is the default, but other models can be used using the <code data-backticks="1">--version</code> or <code data-backticks="1">--v</code> parameter or by using the <code data-backticks="1">/settings</code> command and selecting a model version. Different models excel at different types of images.</h4>
<section class="infoBox">
<div class="title"></div>
<div class="content">
<p><code data-backticks="1">--version</code> accepts the values 1, 2, 3, and 4.<br /><code data-backticks="1">--version</code> can be abbreviated <code data-backticks="1">--v</code></p>
</div>
</section>
<hr />
<h2 id="current-model">Current Model</h2>
<p>The Midjourney V4 model is an entirely new codebase and brand-new AI architecture designed by Midjourney and trained on the new Midjourney AI supercluster. The latest Midjourney model has more knowledge of creatures, places, objects, and more. It's much better at getting small details right and can handle complex prompts with multiple characters or objects. The Version 4 model supports advanced functionality like image prompting and multi-prompts.</p>
<p>This model has very high <span class="popover__title" data-glossary="Coherency" data-glossaryid="b5e87e6c-e1b8-48fc-a424-fbc3b7f7db23">Coherency</span> and excels with <a href="https://docs.midjourney.com/image-prompts">Image Prompts</a>.</p>

<h3 id="version-4-styles-4a-4b-and-4c">Version 4 Styles 4a, 4b, and 4c</h3>
<p>Midjourney Model Version 4 has three slightly different "flavors" with slight tweaks to the stylistic tuning of the model. Experiment with these versions by adding <code data-backticks="1">--style 4a</code>, <code data-backticks="1">--style 4b</code>, or <code data-backticks="1">--style 4c</code> to the end of a V4 prompt.</p>
<p><code data-backticks="1">--v 4 --style 4c</code> is the current default and does not need to be added to the end of a prompt.</p>
<section class="infoBox">
<div class="title">Note on Style 4a and 4b</div>
<div class="content">
<p><code data-backticks="1">--style 4a</code> and <code data-backticks="1">--style 4b</code> only support 1:1, 2:3, and 3:2 aspect ratios.<br /><code data-backticks="1">--style 4c</code> support aspect ratios up to 1:2 or 2:1.</p>
</div>
</section>

<h2 id="niji-model">Niji Model</h2>
<p>The <code data-backticks="1">niji</code> model is a collaboration between Midjourney and <a href="https://spellbrush.com/" target="_blank" rel="noopener">Spellbrush</a> tuned to produce anime and illustrative styles. The <code data-backticks="1">--niji</code> model has vastly more knowledge of anime, anime styles, and anime aesthetics. It's excellent at dynamic and action shots and character-focused compositions in general.</p>
<p>prompt example: <code data-backticks="1">/imagine prompt</code> <code data-backticks="1">vibrant California poppies --niji</code></p>

<h2 id="test-models">Test Models</h2>
<p>Occasionally new models are released temporarily for community testing and feedback. There are currently two available test models: <code data-backticks="1">--test</code> and <code data-backticks="1">--testp</code>, which can be combined with the <code data-backticks="1">--creative</code> parameter for more varied compositions.</p>
<p>prompt example: <code data-backticks="1">/imagine prompt</code> <code data-backticks="1">vibrant California poppies --testp --creative</code></p>

<section class="infoBox">
<div class="title">Notes on current test models <code data-backticks="1">--test</code> and <code data-backticks="1">--testp</code></div>
<div class="content">
<p>Test models only support <code data-backticks="1">--stylize</code> values between 1250&ndash;5000.<br />Test models do not support multi-prompts or image-prompts<br />Test models have a maximum aspect ratio of 3:2 or 2:3.<br />Test models only generate two initial grid images when the aspect ratio is 1:1.<br />Test models only generate one initial grid image when the aspect ratio is not 1:1.<br />Words near the front of the prompt may matter more than words near the back.</p>
</div>
</section>
<hr />
<h2 id="how-to-switch-models">How to Switch Models</h2>
<h3 id="use-the-version-or-test-parameter">Use the Version or Test Parameter</h3>
<p>Add <code data-backticks="1">--v 1</code>, <code data-backticks="1">--v 2</code>, <code data-backticks="1">--v 3</code>, <code data-backticks="1">--v 4</code>, <code data-backticks="1">--v 4 --style 4a</code>, <code data-backticks="1">--v4 --style 4b</code> <code data-backticks="1">--test</code>, <code data-backticks="1">--testp</code>, <code data-backticks="1">--test --creative</code>, <code data-backticks="1">--testp --creative</code> or <code data-backticks="1">--niji</code> to the end of your prompt.</p>
<div data-tomark-pass=""><img src="https://cdn.document360.io/3040c2b6-fead-4744-a3a9-d56d621c6c7e/Images/Documentation/MJ_Parameter-Version.gif" alt="Animated Gif showing how the Midjourney version parameter is typed" data-tomark-pass="" data-lightbox="box" /></div>
<br data-tomark-pass="" />
<h3 id="use-the-settings-command">Use the Settings Command</h3>
<p>Type <code data-backticks="1">/settings</code> and select your preferred version from the menu.</p>
