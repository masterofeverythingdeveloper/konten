<h1>Seeds</h1>
</div>
<hr /></div>
<div class="content_block_text" dir="auto">
<h4>The Midjourney bot uses a seed number to create a field of visual noise, like television static, as a starting point to generate the initial image grids. Seed numbers are generated randomly for each image but can be specified with the <code data-backticks="1">--seed</code> or <code data-backticks="1">--sameseed</code> parameter. Using the same seed number and prompt will produce similar ending images.</h4>
<section class="infoBox">
<div class="title"></div>
<div class="content">
<p><code data-backticks="1">--seed</code> accepts whole numbers 0&ndash;4294967295.<br /><code data-backticks="1">--seed</code> values only influence the initial image grid.<br /><code data-backticks="1">--seed</code> values <a href="https://docs.midjourney.com/models">Model Versions</a> <code data-backticks="1">1</code>, <code data-backticks="1">2</code>, <code data-backticks="1">3</code>, <code data-backticks="1">test</code>, and <code data-backticks="1">testp</code> are non-deterministic and will produce similar, not identical, images.<br />Using the same prompt+seed+parameters in <a href="https://docs.midjourney.com/models">Model Versions</a> <code data-backticks="1">4</code> and <code data-backticks="1">niji</code> will produce identical images.</p>
</div>
</section>
<hr />

<h2 id="sameseed-parameter">Sameseed Parameter</h2>
<p><code data-backticks="1">--seed</code> values create a single large random noise field applied across all images in the initial grid. When <code data-backticks="1">--sameseed</code> is specified, all images in the initial grid use the same starting noise and will produce very similar generated images.</p>
<section class="infoBox">
<div class="title"></div>
<div class="content">
<p><code data-backticks="1">--sameseed</code> accepts whole numbers 0&ndash;4294967295.<br /><code data-backticks="1">--sameseed</code> is only compatible with <a href="https://docs.midjourney.com/models">Model Versions</a> <code data-backticks="1">1</code>, <code data-backticks="1">2</code>, <code data-backticks="1">3</code>, <code data-backticks="1">test</code>, and <code data-backticks="1">testp</code>.</p>
</div>
</section>

<h2 id="how-to-find-a-jobs-seed-number">How to Find a Job's Seed Number</h2>
<h3 id="use-a-discord-emoji-reaction">Use a Discord Emoji Reaction</h3>
<p>Find the seed number of a <span class="popover__title" data-glossary="Job" data-glossaryid="4ca91c3e-a953-4ca4-a4c9-03fff744ef98">Job</span> in discord by reacting with an ✉️ envelope emoji to a Job.</p>
<p><img src="https://cdn.document360.io/3040c2b6-fead-4744-a3a9-d56d621c6c7e/Images/Documentation/MJ_JobID_EmojiReact.gif" alt="Animated Gif showing how to use the Emoji React with an Envelope in Discord to find a Job's seed number" data-lightbox="box" /></p>
<p></p>
<h3 id="use-the-show-command-to-bring-back-old-jobs">Use The Show Command to Bring Back Old Jobs</h3>
<p>To get the seed number for a past image, <a href="https://docs.midjourney.com/docs/show-job">copy the job ID </a> and use the <code data-backticks="1">/show &lt;Job ID #&gt;</code> command with that ID to revive the Job. You can then react to the newly regenerated Job with an ✉️ envelope emoji.</p>
<hr />
<h2 id="how-to-change-seed-numbers">How To Change Seed Numbers</h2>
<h3 id="use-the-seed-or-sameseedparameter">Use the <code data-backticks="1">--seed</code> or <code data-backticks="1">--sameseed</code>Parameter</h3>
<p>Add <code data-backticks="1">--seed &lt;value&gt;</code> or <code data-backticks="1">--sameseed &lt;value&gt;</code> to the end of your prompt.</p>
