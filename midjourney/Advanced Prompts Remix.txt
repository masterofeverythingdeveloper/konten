<h1>Remix</h1>
</div>
<hr /></div>
<div class="content_block_text" dir="auto">
<h4>Use Remix Mode to change prompts, parameters, model versions, or aspect ratios between variations. Remix will take the general composition of your starting image and use it as part of the new Job. Remixing can help change the setting or lighting of an image, evolve a subject, or achieve tricky compositions.</h4>
<section class="infoBox">
<div class="title"></div>
<div class="content">
<p><code data-backticks="1">Remix</code> is an experimental feature that may change or be removed at any time.</p>
</div>
</section>
<hr />
<h2 id="using-remix">Using Remix</h2>
<p>Activate Remix mode with the <code data-backticks="1">/prefer remix</code> command or by using the <code data-backticks="1">/settings</code> command and toggling the <span class="discordbtn" data-tomark-pass=""><code data-tomark-pass="">🎛️ Remix Mode</code></span> button. Remix changes the behavior of the variation buttons (V1, V2, V3, V4) under image grids. When Remix is enabled, it allows you to edit your prompt during each variation. To Remix an upscale select <span class="discordbtn" data-tomark-pass=""><code data-tomark-pass="">🪄 Make Variations</code></span>.</p>
<section class="infoBox">
<div class="title"></div>
<div class="content">
<p>When <code data-backticks="1">Remix</code> is enabled, Variation buttons turn green when used instead of blue.<br />You can switch <a href="https://docs.midjourney.com/models">Model Versions</a> when using Remix.<br />When you are done with Remix use the <code data-backticks="1">/settings</code> or <code data-backticks="1">/prefer remix</code> command to turn it off.<br />Create a standard image variation when Remix is active by not modifying the prompt in the pop-up window.</p>
</div>
</section>

<h3 id="using-parameters-with-remix">Using Parameters with Remix</h3>
<p>You can add or remove <a href="https://docs.midjourney.com/parameter-list">Parameters</a> while using Remix mode, but you must use valid parameter combinations. Changing <code data-backticks="1">/imagine prompt</code> <code data-backticks="1">illustrated stack of pumpkins --version 3 --stylize 10000</code> to <code data-backticks="1">illustrated stack of pumpkins --version 4 --stylize 10000</code> will return an error because Midjourney Model Version 4 is incompatible with the Stylize parameter.</p>

<h2 id="how-to-activate-remix">How to Activate Remix</h2>
<h3 id="use-the-settings-command">Use the Settings Command</h3>
<p>Type <code data-backticks="1">/settings</code> and select <code data-backticks="1">Remix</code> from the pop-up.<br /><span class="discordbtn" data-tomark-pass=""><code data-tomark-pass="">🎛️ Remix</code></span></p>
<h3 id="use-the-prefer-remix-command">Use the Prefer Remix Command</h3>
<p>Type <code data-backticks="1">/prefer remix</code> to toggle Remix mode on and off.<br /><img src="https://cdn.document360.io/3040c2b6-fead-4744-a3a9-d56d621c6c7e/Images/Documentation/MJ_preferRemix.png" alt="Image showing the Midjourney Prefer Remix Command" data-lightbox="box" width="800px" /></p>
