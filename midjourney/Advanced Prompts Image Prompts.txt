<h1>Image Prompts</h1>
</div>
<hr /></div>
<div class="content_block_text" dir="auto">
<h4>You can use images as part of a prompt to influence a Job's composition, style, and colors. Images prompts can be used alone or with text prompts&mdash;experiment with combining images with different styles for the most exciting results.</h4>
<p>To add images to a prompt, type or paste the web address where the image is stored online. The address must end in an extension like .png, .gif, or .jpg. After adding image addresses, add any additional text and parameters to complete the prompt.</p>
<p><img src="https://cdn.document360.io/3040c2b6-fead-4744-a3a9-d56d621c6c7e/Images/Documentation/MJ%20Prompt.png" alt="Image showing the Midjourney prompt structure." data-lightbox="box" width="900" /></p>
<p></p>
<section class="infoBox">
<div class="title"></div>
<div class="content">
<p>Image prompts go at the front of a prompt.<br />Prompts must have two images or one image and additional text to work.<br />An image URL must be a direct link to an online image.<br />In most browsers, right-click or long-press an image and select Copy Image Address to get the URL.<br />The <a href="https://docs.midjourney.com/blend"><code data-backticks="1">/blend</code> command</a> is a simplified image prompting process optimized for mobile users.</p>
</div>
</section>
<hr />
<h2 id="upload-an-image-to-discord">Upload an image to Discord</h2>
<p>To use a personal image as part of a prompt, upload it to Discord. To upload an image, click the <strong>Plus</strong> sign next to where messages are typed. Select <strong>Upload a File</strong>, select an image, and send the message. To add this image to a prompt, begin typing <code data-backticks="1">/imagine</code> as usual. After the prompt box appears, drag the image file into the prompt box to add the image's URL. Alternatively, right-click the image, select <strong>Copy Link</strong>, and then paste the link within the prompt box.</p>
<p><img src="https://cdn.document360.io/3040c2b6-fead-4744-a3a9-d56d621c6c7e/Images/Documentation/Discord_FHZfwDLhLY.gif" alt="Discord_FHZfwDLhLY.gif" data-lightbox="box" /></p>
<p></p>
<section class="infoBox">
<div class="title">Privacy Notes</div>
<div class="content">
<p>Upload images in your <span class="popover__title" data-glossary="Direct Message" data-glossaryid="f826e751-6db0-464b-a6c6-191ed45fe7fe">Direct Message</span>s with the Midjourney Bot to prevent other server users from seeing an image.<br />Image prompts are visible on the Midjourney website unless a user has Stealth Mode.</p>
</div>
</section>

<h2 id="image-prompts-in-v3">Image Prompts in V3</h2>
<p>Image prompts in earlier <a href="https://docs.midjourney.com/models">Midjourney Model Versions</a> are more abstract inspirations than in current models. An image prompt is not the same as building on top of (or "initializing" from) a starting input image. Image prompts are used as visual inspiration, not as a starting point.</p>
<h3 id="image-weight-iw">Image Weight, --iw</h3>
<p>Earlier <a href="https://docs.midjourney.com/models">Midjourney Model Versions</a> can use the image weight parameter <code data-backticks="1">--iw</code> to adjust the importance of the image URLs vs. the text. <code data-backticks="1">--iw 0.25</code> is the default and is used when no <code data-backticks="1">--iw</code> is specified. Higher <code data-backticks="1">--iw</code> values mean the image prompt will have more impact on the finished job.<br />See the <a href="https://docs.midjourney.com/multi-prompts">Multi Prompts</a> page for more information about the relative importance between parts of a prompt.</p>
<p><code data-backticks="1">--iw</code> accepts integers -10,000&ndash;10,000.</p>
<p></p>
