<hr />
<h2 id="custom-preferences">Custom Preferences</h2>
<p>Create custom options using prefer commands to add commonly used parameters to the end of prompts automatically.<br /><code data-backticks="1">/prefer auto_dm</code> Completed Jobs are automatically sent to <span class="popover__title" data-glossary="Direct Message" data-glossaryid="f826e751-6db0-464b-a6c6-191ed45fe7fe">Direct Message</span><br /><code data-backticks="1">/prefer option</code> Create or manage a custom option.<br /><code data-backticks="1">/prefer option list</code> View your current custom options.<br /><code data-backticks="1">/prefer suffix</code> specify a suffix to add to the end of every prompt.</p>
<hr />
<h3 id="prefer-option">Prefer Option</h3>
<p><code data-backticks="1">/prefer option set &lt;name&gt; &lt;value&gt;</code> Creates a custom parameter that you can use to add multiple parameters to the end of prompts quickly.</p>
<p><img src="https://cdn.document360.io/3040c2b6-fead-4744-a3a9-d56d621c6c7e/Images/Documentation/PreferOptionSet.png" alt="PreferOptionSet.png" data-lightbox="box" /></p>
<p><code data-backticks="1">/prefer option set</code> <code data-backticks="1">mine</code> <code data-backticks="1">--hd --ar 7:4</code> creates an option called "mine" that translates to <code data-backticks="1">--hd --ar 7:4</code>.<br /><br data-tomark-pass="" /></p>
<p><img src="https://cdn.document360.io/3040c2b6-fead-4744-a3a9-d56d621c6c7e/Images/Documentation/PreferOptionSet_Used.jpg" alt="PreferOptionSet_Used.jpg" data-lightbox="box" /></p>
<p>Using <code data-backticks="1">/imagine prompt</code> <code data-backticks="1">vibrant California poppies --mine</code>, is interpreted as <code data-backticks="1">/imagine prompt</code> <code data-backticks="1">vibrant California poppies --hd --ar 7:4</code>.</p>
<p>Leave the "value" field empty to delete an option.<br /><br data-tomark-pass="" /></p>
<p><code data-backticks="1">/prefer option list</code> list all options created with <code data-backticks="1">prefer option set.</code> Users can have up to 20 custom options.</p>
<p><img src="https://cdn.document360.io/3040c2b6-fead-4744-a3a9-d56d621c6c7e/Images/Documentation/PreferOptionList.png" alt="PreferOptionList.png" data-lightbox="box" width="400px" /></p>
<p>To delete a custom option, use <code data-backticks="1">/prefer option set</code> <code data-backticks="1">&lt;name to delete&gt;</code> and leave the value field blank.</p>
<hr />
<h3 id="prefer-suffix">Prefer Suffix</h3>
<p><code data-backticks="1">/prefer suffix</code> automatically appends the specified suffix after all prompts. <strong>Use the command without a value to reset.</strong></p>
<p>Command example: <code data-backticks="1">/prefer suffix</code> <code data-backticks="1">--uplight --video</code></p>
