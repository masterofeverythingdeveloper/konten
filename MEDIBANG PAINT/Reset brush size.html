Reset brush size

When changing the brush size from the brush control, you can restore the default brush size by clicking the brush preview window.

This function is useful when you want to temporarily change the brush size.

Let’s try it

Please select the appropriate brush and click “Brush preview window (2)” once.

First, let’s change the brush size from “Brush control (1)”.

When you change the brush size, the brush size of the brush window changes to red letters, and the “*” mark is added to the upper left of the brush preview.

Next, click “Brush Preview (2)”. The brush size returns to its original value.

Tips

This original number is called the default size of the brush.

To change the default size of the brush, you need to change the brush size from the brush edit window. (For details, see How to change the default brush size.)
