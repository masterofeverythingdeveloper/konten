About options before saving

When saving a file other than “MDP format” when saving the canvas, “Save setting” window may be opened before saving.

This section explains save settings for each image format.

PNG format

Transparent PNG
Save it with the transparent (translucent) part on the canvas held.
24-bit PNG
Save the transparent (translucent) part of the canvas without holding it. The file size is a little smaller than the transparent PNG.
Grayscale PNG
The canvas is represented only by white, gray and black and is saved.

JPEG format

Quality
The larger the number is, the better the image quality will be saved, but the file size will be correspondingly larger.
POINT

JPEG format is “lossy compression”. You can not return the degraded image quality later.

TIFF format

Select “Compress” to reduce the file size.

WebP format

Quality
The larger the number is, the better the image quality will be saved, but the file size will be correspondingly larger.
Lossless compression
When checked, it is losslessly compressed and saved.
When checked, save it with the transparent (translucent) part on the canvas held.
